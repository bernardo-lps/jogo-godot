<h1>Código da Liberdade</h1>

<p>
  Uma aventura de fantasia e quebra-cabeças focada na Cifra de César, desenvolvida na Godot Engine.
</p>

<hr>

<h2>Como Jogar</h2>
<p>
  O jogo é dividido em <strong>3 fases</strong>. O mecanismo principal é a decodificação de mensagens usando a <strong>Cifra de César</strong> (uma técnica que desloca as letras do alfabeto).
</p>
<ul>
  <li><strong>Fase 1 - A Vila:</strong> O ancião apresenta o primeiro desafio em um ambiente seguro.</li>
  <li><strong>Fase 2 - A Floresta:</strong> Os desafios se tornam mais complexos enquanto você explora o desconhecido.</li>
  <li><strong>Fase 3 - O Desafio Final:</strong> Teste suas habilidades para salvar a vila.</li>
</ul>
<p>
  Em cada etapa, você deve descobrir a chave de deslocamento correta para traduzir a mensagem cifrada e avançar.
</p>

<h2>Instalação e Execução</h2>
<p>
  Este projeto foi feito para ser executado na Godot Engine.
</p>
<ol>
  <li>Certifique-se de ter a Godot Engine instalada em sua máquina.</li>
  <li>Clone este repositório ou baixe os arquivos.</li>
  <li>Abra a Godot e clique em "Importar".</li>
  <li>Navegue até a pasta do projeto e selecione o arquivo <code>project.godot</code>.</li>
  <li>Após importar, abra o projeto e execute a cena principal para começar a jogar.</li>
</ol>
